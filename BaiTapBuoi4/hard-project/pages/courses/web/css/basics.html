Khái niệm

  • Bộ chọn (selector) giúp xác định phần tử HTML cần định dạng.
  • Cú pháp CSS gồm bộ chọn, dấu ngoặc nhọn và các thuộc tính - giá trị.
  • Độ ưu tiên (specificity) quyết định quy tắc nào được áp dụng khi trùng lặp.
  • Có thể viết CSS nội tuyến, nội bộ hoặc tách file riêng.

Bước thực hành nhanh

  1. Tạo file style.css trong thư mục dự án.
  2. Liên kết file CSS vào HTML bằng thẻ <link> trong phần <head>.
  3. Mở trình duyệt để kiểm tra kết quả hiển thị.
